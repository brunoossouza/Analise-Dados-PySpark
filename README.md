# Análise de Dados com PySpark: Leitura, Escrita e Transformação

Este projeto demonstra a utilização do **Apache Spark (PySpark)** no ambiente Google Colab para o processamento de grandes volumes de dados. O foco principal é a manipulação de diferentes formatos de arquivos e a estruturação de dados.

## 🛠️ Tecnologias Utilizadas
* **Python** (Linguagem principal)
* **Apache Spark (PySpark)** (Processamento distribuído)
* **Google Colab** (Ambiente de desenvolvimento)
* **Formatos:** CSV e Apache Parquet

## 📋 Funcionalidades do Projeto
1.  **Configuração do Ambiente:** Instalação e inicialização do `pyspark`.
2.  **Leitura de Dados:** Importação de arquivos CSV (`videos-stats.csv` e `comentários.csv`) com inferência de schema.
3.  **Exploração de Dados:** Visualização de estruturas (`printSchema`) e amostragem de registros.
4.  **Escrita Otimizada:** Conversão e salvamento dos dados processados para o formato **Parquet**, garantindo maior performance e compressão.

## 📂 Estrutura do Repositório
* `3-LeituraEstruturaArquivo.ipynb`: Notebook com todo o código desenvolvido.
* `videos-stats.csv`: Dataset original com estatísticas de vídeos.
* `comentários.csv`: Dataset com comentários relacionados.

## 🚀 Como Executar
Você pode abrir o notebook diretamente no Google Colab através do link disponível no arquivo `.ipynb` deste repositório para replicar as análises e testes.
